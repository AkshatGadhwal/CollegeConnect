College Connect was brought up as a vision, which aims to be a one stop shop for all the college needs irrespective of college and course.

It aims to create an all in one helpful platform for all college students.

The app uses Material UI for an attractive and appealing interface which attracts user to stay on the app for more time.

This idea was developed for Intra Society Ideathon where it was presented as an Application for College Students.

The app is in currently in development phase.

It's is being build by Saksham Pruthi and Sajal Jain, both pursuing B.Tech
